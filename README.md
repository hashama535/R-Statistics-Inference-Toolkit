# R-Statistics-Inference-Toolkit
Precision Statistical Analysis & Predictive Modeling in RWelcome to my professional statistical portfolio. This repository serves as a demonstration of advanced statistical computational techniques developed during my BS-Statistics (4th Semester) at the College of Statistical Sciences (CSS), University of the Punjab.🚀 Key Technical FeaturesMatrix-Based Regression: Manual implementation of Ordinary Least Squares (OLS) using the fundamental matrix equation:$$b = (X'X)^{-1}X'y$$This ensures a deep understanding of coefficients and intercept derivation beyond standard software wrappers.Statistical Inference & Prediction: Custom logic for calculating 95% Confidence Intervals for the mean and Prediction Intervals for individual values, including manual calculation of Standard Error and T-critical values.Partial Correlation Analysis: Quantifying the unique relationship between variables (e.g., $r_{12.3}$) while controlling for confounding factors.Model Diagnostics: Extraction and interpretation of $R^2$, Residual Standard Error, and Error Variance to assess model fit.📂 Repository StructureFile NameDescription01_Linear_Regression_Builtin.RWorkflow for model fitting, diagnostics, and summary interpretation.02_Multiple_Regression_Matrix_Math.RManual calculation of estimators using Matrix Algebra.03_Statistical_Inference_Intervals.RHypothesis testing and interval estimation for research precision.04_Partial_Correlations.RQuantifying unique variable contributions in complex datasets.💼 Contact & CollaborationName: Malik Hashaam AsifRoll Number: 17-24Institution: College of Statistical Sciences (CSS), University of the Punjab
