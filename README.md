
#  R-Statistics-Inference-Toolkit
### **Precision Statistical Analysis & Predictive Modeling in R**

Welcome to my professional statistical portfolio. This repository serves as a demonstration of advanced statistical computational techniques developed during my **BS-Statistics at the **College of Statistical Sciences (CSS), University of the Punjab**.

##  Key Technical Features

* **Matrix-Based Regression**: Manual implementation of Ordinary Least Squares (OLS) using the fundamental matrix equation:
    $$b = (X'X)^{-1}X'y$$
    This ensures a deep understanding of coefficients and intercept derivation beyond standard software wrappers.

* **Statistical Inference & Prediction**: Custom logic for calculating **95% Confidence Intervals** for the mean and **Prediction Intervals** for individual values, including manual calculation of Standard Error and T-critical values.

* **Partial Correlation Analysis**: Quantifying the unique relationship between variables (e.g., $r_{12.3}$) while controlling for confounding factors.

* **Model Diagnostics**: Extraction and interpretation of $R^2$, Residual Standard Error, and Error Variance to assess model fit.
